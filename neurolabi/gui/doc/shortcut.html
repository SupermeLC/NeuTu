Shortcut Keys

(Cmd refered to as Ctrl for PC)

General

Cmd+ZUndo
Cmd+Shift+ZRedo
Backspace/DeleteDelete selected objects

2D Window

Cmd + GTurn on adding SWC node
Cmd + RTurn on painting
Cmd + ETurn on erasing
[Shift] AMove image or swc node to left. Hold Shift for fast movement.
[Shift] DMove image or swc node to right
[Shift] WMove image or swc node up
[Shift] SMove image or swc node down
=/Up_ArrowZoom in
-/Down_ArrowZoom out
Left_ArrowDecrease Z position
Right_ArrowIncrease Z position
< Decrease radius
> Increase radius

3D Window

Cmd + SSave SWC
